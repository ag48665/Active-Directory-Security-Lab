# Successful Logons Detection

## Description

This detection identifies successful authentication events recorded in Windows Security Logs using Event ID 4624.

Monitoring successful logons helps security analysts understand user activity, identify unusual authentication patterns, validate account usage, and support incident investigations.

---

## Data Source

* Windows Security Event Logs
* Event ID: 4624
* Source: WinEventLog:Security

---

## Detection Logic

### SPL Query

```spl
index=* EventCode=4624
```

---

## MITRE ATT&CK Mapping

| Technique ID | Technique      |
| ------------ | -------------- |
| T1078        | Valid Accounts |

---

## Investigation Guidance

When reviewing successful logon events, analysts should verify:

* Account name
* Source workstation
* Logon type
* Logon time
* Frequency of authentication activity
* Unusual login locations or systems

Potential indicators of suspicious activity include:

* Logins outside normal business hours
* Unexpected administrative account usage
* Authentication from unfamiliar systems
* Multiple successful logons following failed attempts

---

## Expected Result

The search returns successful authentication events generated by Windows systems.

Analysts can use these events to establish user activity timelines and identify potential account misuse.

---

## Validation

The detection was validated using Windows Security Event ID 4624 events collected in Splunk Enterprise.
